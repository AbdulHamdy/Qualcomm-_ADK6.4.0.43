Enumeration Type Documentation

◆ hydra_log_level

enum hydra_log_level

Hydra Log Verbosity Levels.

Enumerator
HYDRA_LOG_ERROR 

errors

Likely only used by panic, assert etc.

Do not abuse this log level. Logs at this level can not be switched off dynamically (indeed the level comparison is optimised out).

HYDRA_LOG_WARN 

warnings

The unusual, the unexpected but not fatal.

HYDRA_LOG_NORMAL 

major, but normal system level events

HYDRA_LOG_VERBOSE 

minor system level events, major module level events

HYDRA_LOG_V_VERBOSE 

minor module level events

HYDRA_LOG_V_V_VERBOSE 

minutiae

HYDRA_LOG_DEFAULT 

default/startup log level.

The level that a system boots into.
