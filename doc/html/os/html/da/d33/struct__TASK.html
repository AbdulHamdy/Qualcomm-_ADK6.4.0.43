Detailed Description

TASK - The information for a system task.

TASKs defined at compile time (static tasks) are held in an array tasks[] created by autogeneration magic. The scheduler holds all defined TASKs (static and dynamic) in a linked list corresponding to the task priority.

Field Documentation

◆ flush_msg

MSG_FLUSH_HANDLER _TASK::flush_msg

Function to flush a message received for the task when the task is up for deletion.

If this function is not provided, the scheduler will try to reclaim the message body (if not NULL) of any queued messaged when deleting the task. This could potentially result in a memory leak if there are other allocated memory chunks in the message body.
