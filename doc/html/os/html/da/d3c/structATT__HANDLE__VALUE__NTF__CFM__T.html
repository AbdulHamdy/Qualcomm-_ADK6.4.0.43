Detailed Description

Confirmation of Handle Value Notification allong with handle and application context received in the ATT_HANDLE_VALUE_NTF_T.

This event shall be generated to indicate ATT_RESULT_SUCCESS or an error code for the attempt to sent the notification to the client.
