Detailed Description

hardware abstraction layer for auxiliary input and output

Functions to control and monitor auxiliary IO pins. Up to 32 PIOs (programmable input/outputs) can be controlled, although they may not all be accessible on a particular hardware platform.

See pio.h for a detailed explanation of the PIO hardware on a Hydra chip.
