Detailed Description

L2CAP data read response.

Note:

  • If L2CAP 1.2 retransmission or flow control is used, it's necessary for the upper layer to respond to data reads to keep the flow control going. Note that the upper layers must read all the data, and that this read can only be successfull (we can't do anything about failures anyway at this point)
  • For Credit Based Flow Control mode on LE L2CAP COC to function appropriately, application shall provide "packets" value in L2CA_DATAREAD_RSP_T same as that notified via "packets" value in preceeding L2CA_DATAREAD_IND_T.
