Detailed Description

DM LE primitives for master or slave to accept/reject connection update param req.

Provides an indication to application to accept/reject connection parameters update request from a peripheral device which has used either the Link Layer or L2CAP update procedure.

In the case of a Link Layer initiated update, the L2CAP Signal ID shall be set to 0.

RETURNS A RSP
