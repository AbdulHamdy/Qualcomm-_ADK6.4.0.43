Detailed Description

L2CAP raw data request.

Send raw L2CAP data to a particular CID/ACL. You should never use this feature unless you're debugging L2CAP itself or doing low-level testing!

Raw data writes will generate normal L2CA_DATAWRITE_CFM_T primitives.
