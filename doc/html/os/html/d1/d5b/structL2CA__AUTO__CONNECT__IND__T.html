Detailed Description

Automatic connection indication.

L2CA_AUTO_CONNECT_IND which used on the responder side when a peer wants to initiate a connection. There are no special parameters for the auto-connect indication. L2CAP expects a L2CA_AUTO_CONNECT_RSP which allows the application to either accept or reject the connection.
