Detailed Description

Handle Value Notification or Indication from the server.

In case of Notification (flags contain bit ATT_HANDLE_VALUE_INDICATION) the client shall respond using ATT_HANDLE_VALUE_RSP_T that it has received the Notification.

In case of Indication the client do not need to response back.
