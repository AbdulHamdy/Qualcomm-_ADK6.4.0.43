Detailed Description

Server Connection Response.

This event allows the higher layers to specify its parameters for a new DLC connection initiated by the peer device. It is sent in response to a RFC_SERVER_CONNECT_IND.

There are currently no flag options available for this prim and thus the flags field should be set to 0.

l2ca_controller_t RFC_SERVER_CONNECT_RSP_T::local_l2cap_control

Local L2CAP controller ID Initial MSC parameters. See Modem Status Request for details of modem and break signals. msc_timeout is in Milliseconds and is the time allowed for the initial (automatic) handshaking of the MSC commands immediately after a connection has been established and before any data is sent. Should the timeout fire then this implies a badly behaving peer and the action will be to assume the handshake has completed.
