◆ bg_int_fn

typedef void(* bg_int_fn) (void **priv)

Background interrupt handler type.

Parameters
privPointer to the partner task's private memory area. This must be ignored if the bg int does not have a designated partner task.

Note that CarlOS-style background interrupt handlers differ slightly in form and function from Audio-style handlers. Firstly, they don't take the private memory pointer, since by definition they are not partnered with a task; secondly, they have no notion of multiplexing of logical bg ints - they are either raised or not - and as a result, thirdly, they expect the scheduler to clear their flag and hence don't do it themselves. Hence the CarlOS-style autogeneration macros create a wrapper of type bg_int_fn which handles these interface differences and then calls the real handler.

typedef struct _TASK TASK

TASK - The information for a system task.

TASKs defined at compile time (static tasks) are held in an array tasks[] created by autogeneration magic. The scheduler holds all defined TASKs (static and dynamic) in a linked list corresponding to the task priority.
