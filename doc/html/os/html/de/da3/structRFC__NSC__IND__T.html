Detailed Description

Non Supported Command indication.

Non Supported Command Response. See TS07.10 5.4.6.3.8. Indicates that the remote device did not understand the last frame sent. NB the command_type given is the RFCOMM command type for the command, not one of the RFC_PRIM_T 'type' enumerations.
