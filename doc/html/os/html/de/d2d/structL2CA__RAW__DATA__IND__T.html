Detailed Description

L2CAP raw data indication.

Receive raw L2CAP data. You should never use this feature unless you're debugging L2CAP itself or doing low-level testing! Before you can receive raw data, this must be turned on using L2CA_RAW_MODE_REQ_T. Depending on the raw mode chosen, the data provided here might be just a copy for information, or L2CAP might be expecting the application to deal with any required responses.
