Detailed Description

Send L2CAP data packet confirmation.

Confirmation sent when data has been sent.

The timing of the confirmation is dependant on the channel mode, for basic mode the confirmation is generated locally as soon as the L2CAP packet has been transmitted, for Retranmission and Flow Control modes the confirmation is sent on receiving an acknowledgement from the peer.

See L2CA_DATAWRITE_REQ_T for more details.
