Detailed Description

Send Write Request, Write Command, or Signed Write Command to the server.

Flags is used to determine whether to send Write Request (ATT_WRITE_REQUEST), Write Command (ATT_WRITE_COMMAND) or Signed Write Command (ATT_WRITE_COMMAND | ATT_WRITE_SIGNED).

Signed Write Command can only be sent to bonded server.
