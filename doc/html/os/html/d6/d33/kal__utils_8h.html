◆ tracepoint_trigger_1()

void tracepoint_trigger_1 ( void  )

The first potential trigger point for instruction trace hardware.

This function does nothing except provide an address for triggering starting or stopping the instruction trace hardware.

This function should be present for instruction tracing on Apps p0 and p1.

Warning
Renaming this function should be avoided as pylib depends on it for the start/stop tracing feature. Other subsystems, e.g. Audio, may also be using this name.
