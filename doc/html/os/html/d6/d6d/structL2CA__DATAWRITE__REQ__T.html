Detailed Description

Send L2CAP data packet request.

Request to send L2CAP packet.

Currently only one packet can be sent per channel at a time, higher layers may only transmit another packet when they receive a L2CA_DATAWRITE_CFM_T primitive.

The 'con_ctx' field in the primitive is automatically returned in the corresponding L2CA_DATAWRITE_CFM_T primitive, it can be used by the higher layer to store a context for the L2CAP data packet.
