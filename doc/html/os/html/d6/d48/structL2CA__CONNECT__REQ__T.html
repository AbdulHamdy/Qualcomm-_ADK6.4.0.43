Detailed Description

L2CAP connect request.

These are primitives that are used by L2CAP internally. They must not be sent by the application. Use L2CA_AUTO_CONNECT_ instead.

Connect request to remote device Note the addition of a psm_local - otherwise there is no way of knowing where to send the confirm to.
