◆ _BITARRAY_WORD_LENGTH

#define _BITARRAY_WORD_LENGTH (   length)    (((length + (UINT_BIT - 1)) / UINT_BIT) * sizeof(unsigned))

Extremely simple interface for treating a uint16 array as a bit array.

Note: no attempt made to give these function-call semantics - they're just shorthand. Turn the number of bits into the number of underlying words required to store them.
