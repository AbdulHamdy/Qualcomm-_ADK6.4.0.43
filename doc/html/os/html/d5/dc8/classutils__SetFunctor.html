Detailed Description

Intrusive Set Functor (Abstract Base)

Used as argument to utils_Set_visitEach() allowing a functor (context + function) to be invoked on each member.

This is useful in any situation where a specific context is needed when processing each member. E.g. for implementing aggregate functions (such as sum, max, min).
