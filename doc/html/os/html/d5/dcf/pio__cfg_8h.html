Detailed Description

PIO configuration.

USAGE

This is the public header for the PIO configuration module. It should be included in all modules that need to configure PIOs.

typedef void(* pio_cfg_cb) (pio_cfg_ops op, uint16 bank, uint32 mask, uint32 value)

Typedef for the pio_cfg_op callback.

The op, bank and mask parameters are the ones that were given to pio_cfg_op. The value parameter is used to indicate either an error code or a bit mask of the failed PIOs.

◆ pio_cfg_op()

pio_cfg_status pio_cfg_op ( pio_cfg_ops  op,
uint16  bank,
uint32  mask,
uint32  value,
pio_cfg_cb  cb 
)

Initiates a PIO configuration operation.

Parameters
opOperation ID, see pio_cfg_ops.
bankBank ID.
maskBit mask indicating which PIOs to configure.
valueConfiguration value or bit mask indicating the configuration value per PIO.
cbCallback to be called once the configuration is completed. Some errors may prevent the callback from being called. Please note that in some cases the callback may be called brefore this function returns.
Returns
PIO_CFG_SUCCESS if request was successful, PIO_CFG_NO_BULK_QUERY indicates that the query requested cannot be made for more than one PIO at a time, PIO_CFG_NON_XIOS_IN_MASK indicates an XIO only operation requested for non-XIO PIOs. The callback will be called only if the PIO_CFG_SUCCESS status is returned.
