Detailed Description

Confirmation of Handle Value Notification/Indication.

In case of Indication two ATT_HANDLE_VALUE_CFMs are sent, one when the indication was sent (ATT_RESULT_SUCCESS_SENT), and another when the client confirmed the indication (ATT_RESULT_SUCCESS).
