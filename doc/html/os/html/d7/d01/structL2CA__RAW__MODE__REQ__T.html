Detailed Description

L2CAP raw data mode.

Enable or disable L2CAP raw data mode support for given CID and/or related ACL. You should never use this feature unless you're debugging L2CAP itself or doing low-level testing! When turned on, you will receive data in L2CA_RAW_DATA_IND_T
