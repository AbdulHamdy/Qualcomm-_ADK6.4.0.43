Detailed Description

L2CAP signal MTU violation.

Indicates that an L2CAP signalling MTU violation has occurred. If the application receives this signal, the last signal (possibly a L2CA_CONFIG_REQ) should be resent such that L2CAP can retry with a lower MTU.
