#define UTILS_ROUND_UP_LOG2 (   _value,
  _align_log2 
)    (((_value) + ((1 << (_align_log2)) - 1)) & ~((1 << (_align_log2)) - 1))

Round up the given value to the nearest multiple of 2 to the power of the given exponent.

If the given value is already a multiple it is returned as it is.

This function is useful for alignment calculations. e.g. Round the given value to a 32-bit word boundary (4-bytes): addr_aligned = UTILS_ROUND_UP_LOG2(addr, 2); // 2^2 = 4 bytes = 32 bits

Parameters
[in]_valueThe value to round up.
[in]_align_log2log2 of the multiple to align _value up to.
Returns
_value rounded up to the nearest multiple of 2^ _align_log2.
