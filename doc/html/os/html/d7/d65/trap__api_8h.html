uint16  bank,
uint32  mask,
uint32  value

Main function for doing an IPC PIO operation.

This is used to alter PIO settings which are not available to Apps P1.

Parameters
opIPC PIO operation.
bankPIO bank index.
maskPIO mask.
valueMask representing a parameter for each PIO. The actual parameter depends on the type of operation chosen.
Returns
Mask indicating a pass or fail status for each PIO (0 - Pass, 1 - Fail).
