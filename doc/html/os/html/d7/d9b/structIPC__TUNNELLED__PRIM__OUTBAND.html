Detailed Description

DIR BOTH Arbitrary primitive passed explicitly to or from the customer app.

The primitive itself is opaque as far as this protocol is concerned

Note
The payload is not const because it needs to be possible for the receiver to pfree it (i.e. call ipc_send_pfree())
