Detailed Description

Automatic connection indication on specific transport.

L2CA_AUTO_TP_CONNECT_IND is used on the responder side to indicate an incoming connection request from a peer device on a specific transport. Conftab is reserved for future use. Conftab passed on to the application is owned and freed by the application.
