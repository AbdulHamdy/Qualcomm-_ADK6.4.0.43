Detailed Description

RFCOMM over AMP support.

The following set of commands are available to support the use of AMP over RFCOMM. The RFCOMM layer is behaving purely as a router for these commands and routing from the host to L2CAP and visa versa.

MOVE CHANNEL

Local host Peer RFC_L2CA_MOVE_CHANNEL_REQ —> RFC_L2CA_MOVE_CHANNEL_IND —> <— RFC_L2CA_MOVE_CHANNEL_RSP <— RFC_L2CA_MOVE_CHANNEL_CFM RFC_L2CA_MOVE_CHANNEL_CMP_IND—>

LINK LOSS

<— RFC_L2CA_AMP_LINK_LOSS_IND

AMP Move Channel request

This primitive is used to support an L2CAP move channel request, via rfcomm, issued by the host.
