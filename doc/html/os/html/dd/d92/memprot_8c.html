void memprot_enable ( void  )

Set up memory protection handling.

This function does not have options because callers (i.e. Apps main) aren't meant to worry about the details - memprot knows what to do.

This function sets up the interrupt handler for restricted memory accesses by P1. The default behaviour is to report details of the access into the firmware log and then clear the fault allowing P1 to continue
