volatile uint16 dorm_cfg_t::min_mmu_rate

The minimum rate we will allow for the MMU clock on this processor.

It is in the units required by the hardware: (clock_rate / MHz) = 60 / (1 + min_mmu_rate). 0 is therefore a sensible default.

Note that this value may be set to 0 in the foreground to ensure that the background does not subsequently reduce the clock rate before exiting the sleep code, hence tests for this are intrinsically race prone.
