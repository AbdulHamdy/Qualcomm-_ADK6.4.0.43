Detailed Description

Tell the peer that we are not ready to receive more data.

In retransmission mode it's possible to inform the remote peer that we are currently not able to receive any more data even if the rx/tx window is not yet full. This primitive can toggle the status of this field, but data may still get through due to crossing signals.

This signal is only used in retransmission or enhanced retransmission mode.
