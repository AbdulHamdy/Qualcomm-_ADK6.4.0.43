Detailed Description

Automatic connection confirm on specific transport.

L2CA_AUTO_TP_CONNECT_CFM is used both on the initiator and responder side of a connection to indicate the status of a connection on a specific transport. config_ext is extended configuration values other than L2CA_TP_CONFIG_T, it is reserved for future use. It is advised to free config_ext to make user code future proof (to avoid memory leak).

See also
L2CA_AUTO_CONNECT_CFM_T

BR/EDR connection : L2CA_AUTO_TP_CONNECT_CFM is an extended version of L2CA_AUTO_CONNECT_CFM to accomodate LE connection oriented channels. Except for tp_addrt information, usage of this API for BR/EDR connection would remain the same as L2CA_AUTO_CONNECT_CFM.

LE connection : Application can refer rx credits, local and remote MTU values in L2CA_TP_CONFIG_T.
