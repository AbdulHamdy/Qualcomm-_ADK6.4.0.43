void*(* BUFFER_MSG_META_FNS::peek) (BUFFER_MSG *b, uint16 *len)

Find the meta data entry for the first message.

Parameters
bThe buffer to search meta data
lenLocation to store meta data length
Returns
A pointer to the meta data, or NULL if there is no data. If len is not NULL, then *len is set to the number of bytes of meta data available. The returned pointer is valid until the owning message is cleared.
