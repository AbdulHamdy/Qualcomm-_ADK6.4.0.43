Detailed Description

Connection confirmation.

This event is sent when a new DLC connection has been successfully opened. The actual type used will depend on whether it is a client (outbound) or server (inbound) connection.

flags usage: Bit 0 = 0 , Basic mode being used Bit 0 = 1 , ERTM mode being used
