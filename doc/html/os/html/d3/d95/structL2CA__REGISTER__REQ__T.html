Detailed Description

Register PSM request.

Register a PSM so that L2CAP can multiplex between protocols.

In the case where we are operating under the QTIL scheduler, the phandle is the queue identifier for a particular higher layer protocol. To register a PSM on LE transport use L2CA_REGFLAG_FOR_LE_TP and L2CA_REGFLAG_USE_TP_PRIMS register flags
