◆ configure_sw_interrupt_raw()

swint_id configure_sw_interrupt_raw ( int_level  level,
void(*)(void)  handler 
)

Configure a software interrupt source.

Configure a software interrupt to call "handler" at priority "level". It is not necessary to enable the source separately.

If no software interrupt sources are available then this function will panic. This function is similar to configure_sw_interrupt(), except that it always attempts to allocate a hardware event rather than using a software multiplex for

Returns An identifier for this software interrupt. This should be passed to generate_sw_interrupt() to trigger the interrupt, and may be passed to unconfigure_sw_interrupt() when this interrupt is no longer required.
