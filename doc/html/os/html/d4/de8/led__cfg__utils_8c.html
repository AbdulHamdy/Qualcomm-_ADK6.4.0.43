◆ led_cfg_is_clocked()

bool led_cfg_is_clocked ( void  )

Check if the LED controller is clocked.

If not then we can't access the registers. The clock is turned on by the Curator when a PIO is muxed to the LED controller and turned off in deep sleep if there is no PIO muxed to the LED controller.

Returns
TRUE if LED controller is clocked, FALSE otherwise.
