Detailed Description

Client Connection Request.

This event allows the higher layers to initiate a new outgoing DLC connection, using the auto connect mechanism.

Initial MSC parameters:

See Modem Status Request for details of modem and break signals.

"msc_timeout" is in Milliseconds and is the time allowed for the initial (automatic) handshaking of the MSC commands immediately after a connection has been established and before any data is sent. Should the timeout fire then this implies a badly behaving peer and the action will be to assume the handshake has completed.
