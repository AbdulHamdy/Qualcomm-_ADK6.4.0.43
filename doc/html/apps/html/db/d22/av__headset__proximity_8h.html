#define appProximityClientRegister (   task)    FALSE

Register with proximity to receive notifications.

Parameters
taskThe task to register.
Returns
TRUE if the client was successfully registered. FALSE if registration was unsuccessful or if the platform does not have a proximity sensor. The sensor will be enabled the first time a client registers.
