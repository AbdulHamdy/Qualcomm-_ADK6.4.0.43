Detailed Description

DESCRIPTION
This module works as a container for all private and common data that is used across sink qpplication. It provides API to access the data from outside. It interacts with configuration entities to read/write configuration data. More over this module contains module specific run time data (as a common runtime data for entire sink application), and those data also exposed using defined API's
NOTES Module does not have any intelligence to manipulate the data, or verify the data or its contents ,it is the user to decide hoe to use this data and when to update the data.
