◆ appDeviceGetHandsetAttributes()

bool appDeviceGetHandsetAttributes ( bdaddr *  bd_addr,
appDeviceAttributes *  attributes,
int *  index 
)

Get handset BT address and attributes.

Return the first handset BT address and attributes from the PDL starting at index. Can be used to iterate through the known handset devices with repeated calls incrementing the index parameter.

Passing a NULL as the index parameter will return the first handset instance found.

Parameters
bd_addrBT address of handset device at index.
attributesAttributes of handset device at index.
[in,out]indexPointer to value holding the index. On entry will start searching at this index. On return this holds the index at which the handset was found
Returns
bool TRUE success, FALSE failure, no more handset attributes.

peer_hfp_profile_connectedThe hfp version of the peer's handset.
Returns
uint8 Bitmask of profiles that were known to be supported before this function was called.

This function should only be called if the address is known to be for a standard Bluetooth device (not Bluetooth Low Energy). Where a typed Bluetooth Address is used (typed_bdaddr) the type must be TYPED_BDADDR_PUBLIC, but this is not a sufficient check.
appDeviceTypeIsHandset() is an alternative function that can be used if the device may not have been fully paired yet.
Parameters
bd_addrPointer to read-only device BT address.
Returns
bool TRUE if is a handset and attributes have been exchange, FALSE otherwise.
