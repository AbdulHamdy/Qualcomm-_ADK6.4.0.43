Register with accelerometer to receive notifications.

Parameters
taskThe task to register.
Returns
TRUE if the client was successfully registered. FALSE if registration was unsuccessful or if the platform does not have a accelerometer sensor. The sensor will be enabled the first time a client registers.

◆ appAccelerometerGetDormantConfigureKeyValue()

bool appAccelerometerGetDormantConfigureKeyValue ( dormant_config_key *  key,
uint32 *  value 
)

Obtain the key/value to pass to DormantConfigure to allow the accelerometer interrupt to wake the device from dormant mode. The caller may just pass the returned values into the DormantConfigure trap, or modify them to enable other wake sources on the same key.

Parameters
[out]keyThe key to be passed to DormantConfigure
[OUT]value The value to be passed to DormantConfigure
Returns
TRUE if the accelerometer is able to wake the chip from dormant and the key/value are valid, otherwise FALSE.
