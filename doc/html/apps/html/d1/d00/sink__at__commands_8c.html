Detailed Description


Application level control of AT commands sent and received via the HFP
connections.

These can be configured in 2 ways, received AT commands that ate unrecognised by the HFP library will be passed to the applciation (here)

If these match any of the commands in the cofiguration, then a response also in the configuration will be sent to the AG device.

Alternatively, User events can be configured to send configured AT commands (from the same list) to the AG on receipt of any user event,

E.g. This allows the user to configure an event that can be sent to the AG on connection or when the battery level changes.
