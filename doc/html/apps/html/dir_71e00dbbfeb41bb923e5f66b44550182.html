file  av_headset_temperature.h [code]
 Clients of the temperature module should use this API. If a client just wants to read the current temperature, appTemperatureGet() should be used. If a client wants to be informed about changes in temperature with respect to limits, the client should register using appTemperatureClientRegister. The module will then send status temperatureMessages to the client.
