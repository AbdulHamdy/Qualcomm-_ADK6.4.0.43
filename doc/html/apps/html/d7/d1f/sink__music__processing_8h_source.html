5 FILE NAME
6  sink_music_processing.h

8 DESCRIPTION
9  Module handling music processing settings.
10  Music processing settings consist of music processing mode (see A2DP_MUSIC_PROCESSING_T)
11  and music processing bypass flags.
12 
13  Music processing mode is stored internally in the sink application in different format
14  than it is sent over the air for peers synchronisation purpose.
15 
16  When sent over the air both music processing bypass flags and selected EQ bank are packed together
17  in one 16 bit word. Three least significant bits are selected EQ bank (0 to 6).
18  The most significant bit is a flag indicating that bypass flags should be applied.
19  Bits in between are used to carry bypass music enhancement flags.
20 
21  In the sink application bypass flags and processing mode are stored in two separate variables.
22  Bit positions are not changed, thus they are still not overlapping.
23  However processing mode is full A2DP_MUSIC_PROCESSING_T type instead of just EQ bank number.
24  In effect processing mode is using 4 bits as oppose 3 bits used by EQ bank number.
25 
26 */
