◆ configManagerGetReadOnlyConfig()

uint16 configManagerGetReadOnlyConfig ( uint16  config_id,
const void **  data 
)

configManagerGetReadOnlyConfig

DESCRIPTION
Open a config block as read-only.
The pointer to the memory for the config block is returned in *data.
If the block cannot be opened, this function will panic.
PARAMETERS
config_id [in] Id of config block to open data [out] Will be set with the pointer to the config data buffer if successful, NULL otherwise.
Returns
uint16 size of the opened config block buffer. The size can be 0, e.g. if the config is an empty array.Here size returned is not equivalent to sizeof(), but represents the number of uint16's containing config blocks information.

configManagerGetWriteableConfig

DESCRIPTION
Open a config block as writeable.
The pointer to the memory for the config block is returned in *data.
If the block cannot be opened, this function will panic.
PARAMETERS
config_id [in] Id of config block to open data [out] Will be set with the pointer to the config data buffer if successful, NULL otherwise. size [in] Size of the buffer to allocate for the config data. Set this to 0 to use the size of config block in the config store.
Returns
uint16 Size of the opened config block buffer. The size can be 0, e.g. if the config is an empty array.
