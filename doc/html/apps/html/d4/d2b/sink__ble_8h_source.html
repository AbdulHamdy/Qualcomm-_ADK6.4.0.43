889 DESCRIPTION
890  Handle specific failure cases when CL_DM_BLE_CONNECTION_PARAMETERS_UPDATE_CFM or CL_DM_BLE_CONNECTION_PARAMETERS_UPDATE_IND is received
891  This handling is only used if BLE upgrade is in progress and there is a need to retry the connection parameter
