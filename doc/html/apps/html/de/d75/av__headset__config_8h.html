◆ appConfigBleAddressChangeTime

#define appConfigBleAddressChangeTime ( )    (BLE_RPA_TIMEOUT_DEFAULT)

The time before changing the address used for Bluetooth Low Energy adverts and connections

The earbuds use Resolvable Private Addresses (RPA) for BLE connections. This address must be updated on a timer, the update being handled by the libraries.

Note
When using RPA the device is not allowed to use its real address in any adverts. Responding to own address is allowed.

◆ appConfigBleAllowedOutOfCase

#define appConfigBleAllowedOutOfCase ( )    (FALSE)

Define whether BLE is allowed when out of the case.

Restricting to use in the case only will reduce power consumption and extend battery life. It will not be possible to start an upgrade or read battery information.

Note
Any existing BLE connections will not be affected when leaving the case.
