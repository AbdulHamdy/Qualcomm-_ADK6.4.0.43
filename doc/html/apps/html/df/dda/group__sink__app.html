Detailed Description

Application for use in all devices that play audio or calls.


INTRODUCTION

The sink application is a multi-purpose application that can be configured for use in playing audio, allowing music streams carried over A2DP to be played, whilst providing control of the source device using standard Bluetooth protocols for media control (AVRCP) and call control (HFP).

void handleUsrNFCTagDetected ( Task  task,
Message  message 
)

#include <sink_nfc.c>

Implementation file to handle nfc connection library messages.

This function is called when the NFC RF field has been detected It is provided for ADK4.0 backward compatibility.

ADK 4.x code

NFC tag detected, determine action based on current connection state case EventUsrNFCTagDetected:

The code below should be replaced something similar to sinkNfcHandleTagReadStarted(task) to enable fast connection. This would be triggered on the detection of NFC Field from the phone. The main difference is that the Sink app has NO guarantee that the reader has actually read the tag.

For instance: the end user may just have waved his/her phone next to the tag but yet activated the tag or started reading the content of it.

The TagReadStarted function has the advantage of informing the sink app that the PHONE has almost completed reading the content of the Sink App TAG
