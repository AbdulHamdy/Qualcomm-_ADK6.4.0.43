void appKymeraScoForwardingPause ( bool  pause)

Enable/Disable generation of forwarded SCO audio.

Parameters
[in]pauseIf TRUE will stop generating SCO audio data, if FALSE will resume.

This function leaves the SCO/SFWD chains up and running and flips the switched passthrough consumer between 'consume' and 'passthrough' modes to control if forwarded SCO data generated on the stream source for the SCOFWD packetiser to send.

static void appKymeraSetAecSameIOClockSource ( Operator  aec_op)
static

Message AECREF operator that the back-end of the operator are coming from same clock source. This is for optimisation purpose and it's recommended to be enabled for use cases where speaker input and microphone output are synchronised (e.g. SCO and USB voice use cases). Note: Send/Resend this message when all microphone input/output and REFERENCE output are disconnected.

aec_opThe AEC Reference operator.
