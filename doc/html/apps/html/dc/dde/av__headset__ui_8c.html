Play prompt.

Parameters
promptThe prompt to play.
interruptibleIf TRUE, always play to completion, if FALSE, the prompt may be interrupted before completion.
client_lockIf not NULL, bits set in client_lock_mask will be cleared in client_lock when the prompt finishes - either on completion, when interrupted, or if the prompt is not played at all, because the UI is not currently playing prompts.
client_lock_maskA mask of bits to clear in the client_lock.

Parameters

client_lockIf not NULL, bits set in client_lock_mask will be cleared in client_lock when the tone finishes - either on completion, when interrupted, or if the tone is not played at all, because the UI is not currently playing tones.
client_lock_maskA mask of bits to clear in the client_lock.
