sink_ba_app_mode_t

This type defines Broadcast Audio Application Mode. sink_ba_appmode_normal is the mode where broadcat audio is not available sink_ba_appmode_broadcaster is the mode where BA Broadcaster is Active sink_ba_appmode_receiver is the mode where BA Receiver is Active
