Detailed Description

DESCRIPTION
Common Header File used in broadcast Audio. This header file is used by sink_ba/sink_boradcaster/sink_receiver and sink_ba_plugin. It is used to have common defines and other data which is shared between these sub modules This file is not used to expose any interfaces.
