◆ packet_master_header_length_t

typedef uint32 packet_master_header_length_t(packet_master_t *p, uint32 number_audio_frames)

Get the packet header length in octets.

Parameters
pThe packet instance.
number_audio_framesThe number of audio frames to transport in the packet.
Returns
The header length in octets.

If number_audio_frames is zero, the function will return the number of octets in the main packet header. For non-zero number_audio_frames, the returned value will also include the number of header octets per frame.
