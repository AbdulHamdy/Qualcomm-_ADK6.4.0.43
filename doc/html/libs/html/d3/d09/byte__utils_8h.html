ByteUtilsStrLCpyUnpack.

DESCRIPTION
Unpacks and copies up to dstsize - 1 characters from the string src to dst, NUL-terminating the result if dstsize is not 0.
If the return value is >= dstsize, the output string has been truncated. It is the caller's responsibility to handle this.
Returns
Number of characters we tried to copy (length of src).
