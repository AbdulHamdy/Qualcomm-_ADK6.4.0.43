void AghfpSendError ( AGHFP *  aghfp)

Send
to the remote end.

This function allows the client of the aghfp library to send an AT error response to the remote device. This command is valid as soon as the RFCOMM link is made - ie before a full SLC is established.

Returns a AGHFP_SEND_ERROR_CFM message to the client to indicate whether the data was sent successfully or not.
