GaiaVoiceAssistantSendData.

Send voice data to the (single if any) connected Host supporting VA.

DESCRIPTION
Send voice data to the (single if any) connected Host supporting VA. Returns the number of voice data octets sent.
Note
This function does not have a transport parameter. The implicit GAIA_COMMAND_VA_VOICE_DATA is not acknowledged. The caller is responsible for handling failure.
