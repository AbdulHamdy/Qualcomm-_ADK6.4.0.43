86 /******************************************************************************
87 NAME
88  ahiCanSendTransportData
89 
90 DESCRIPTION
91  Check if there are enough resources to send an outgoing data packet to the
92  Host transport.
93 
94  This function is used to throttle the maximum number of messages that
95  are posted onto the Host transport's Task. Without it, it is possible
96  to run out of VM resources (pmalloc slots) before the Host transport
97  has had a chance to process the queued messages.
98 
99  Throttling messages is only required for operations that send multiple
100  messages to the Host without waiting for a reply for each one.
101  Currently that is:
102  AHI_CONFIG_GET_CFM
103  AHI_CONFIG_FILE_GET_CFM
104 
105 RETURNS
106  TRUE if it is ok to send a data packet, FALSE otherwise.
107 */
