UpgradeHostIFTransportConnect.

DESCRIPTION
Process an upgrade connect request from a client. Send a response to the transport in a UPGRADE_TRANSPORT_CONNECT_CFM.
If Upgrade library has not been initialised return upgrade_status_unexpected_error.
If an upgrade client is already connected return upgrade_status_already_connected_warning.
Otherwise return upgrade_status_success.
