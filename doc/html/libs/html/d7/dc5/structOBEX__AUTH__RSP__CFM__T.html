Detailed Description

Confirmation message on receiving the Authentication response.

The library returns the remote Authentication Response for the Authentication Challenge sent to the remote side. The application can use the MD5 algorithm to validate the response with the challenge.

If application failed to authenticate the received response, it must disconnect the connection using ObexDisconnectRequest().
