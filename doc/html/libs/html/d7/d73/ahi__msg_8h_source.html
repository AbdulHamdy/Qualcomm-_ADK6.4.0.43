5 FILE NAME
6  ahi_msg.h

8 DESCRIPTION
9  AHI internal message types. The types in this header are intended
10  to be used only within the AHI library.
11 
12  When an AHI data packet is received from the host it gets converted
13  to one of the types defined below. However if the the data packet
14  has no payload (header only) then only the opcode is needed and
15  there will be no type defined for it below.
16 
17  Note: The memory layout of an AHI data packet and an internal
18  message is different on BlueCore devices:
19  AHI data packet (ahi_protocol.h):
20  In "stream" format, i.e. one byte per word.
21  AHI internal message (ahi_msg.h):
22  In "packed" format, i.e. two bytes per word.
23 
24 */
