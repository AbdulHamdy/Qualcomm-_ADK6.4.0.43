Detailed Description

This message is sent when a Service Level Connection has been lost and the library is attempting to automatically reconnect. This can be configured in hfp_init_params. If all reconnection attempts fail then HFP_SLC_DISCONNECT_IND will be sent with status hfp_disconnect_link_loss.
