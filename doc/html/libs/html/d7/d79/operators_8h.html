Detailed Description

DESCRIPTION
Kymera operator related helper functions. It consists of two APIs, low level one and higher level one.
First API provides OperatorsCreate() function to create any operator and set of functions to configure operators properties. One function per property.
Second API builds on top of low level one and provide function per operator to create and configure it in single call.
