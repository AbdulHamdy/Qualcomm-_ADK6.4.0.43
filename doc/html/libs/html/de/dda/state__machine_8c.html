Initialise a state machine instance.

Parameters
fsmThe state machine to initialise.
initial_stateInitial state of the state machine. Must not be 0
state_transitionsTable containing all the transitions per state. There must be one entry per state.
state_countNumber of states in the transition table.
default_transitionDefault transition to use for unhandled events if the current state does not have its own default transition. This can be 0, in which case the library default transition is used instead.
user_ctxUser data that can be accessed in action functions.

sm_event_t smPanicAction ( const sm_fsm_t *  fsm)

An action function that always panics.

If logging is enabled it will output some info before it panics.

Note: This is the default transition action for any state machine that does not define its own default transition.
