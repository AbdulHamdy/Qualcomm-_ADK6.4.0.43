gattHandleAttConnectCfm

DESCRIPTION
Handles the ATT_CONNECT_CFM from BlueStack. Pass on the INITIATING state so that the app can 'disconnect' the CID before the connection is established.
Also, unless prepared, an ATT_CONNECT_CFM can be received for an ATT over BR/EDR slave connection and the GATT library won't know what to do with it. In that case Panic in debug and ignore in default lib version.
Returns

gattHandleAttConnectInd

DESCRIPTION
Handles the ATT_CONNECT_IND for initiating a GATT connection
from BlueStack. Pass it straight through, no need to store state.
The CID is locked until application responds to GATT_CONNECT_IND to prevent race condition when using BLE.
