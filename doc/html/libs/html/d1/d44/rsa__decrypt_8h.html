decrypt a block of memory using the supplied public key and the supplied block of memory.

Parameters
startPointer to the start of the signature.
mod
A
Returns
None.

Use the montgomery multiplication algorithm to exponentiate the block modulus the key given. The exponent is always 3, atm. More choice may follow, but not in this version of the firmware.

A is a lump of memory passed in initialised to R^2N mod M. See Knuth for details of the Montgomery multiplication algorithm.
