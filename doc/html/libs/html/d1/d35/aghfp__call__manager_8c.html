uint8  type_number,

const uint8 *  number,
uint16  size_string,
const uint8 *  string 
)

Sets caller id details for +CLIP notifications.

Parameters
aghfpA pointer to the profile instance.
type_numberThis parameter specifies the format of the phone number provided and can be one of the following values: 128-143=Unknown format. 144-159=International number. 160-175=National number. See the description of "+CLIP" in HFP 1.5 spec for more details.
size_numberThe number of characters in the phone number. (Not including any null terminator)
numberA uint8 array containing one digit of the phone number per array element. The digits are in ASCII format and may include special characters such as '+'. The number should NOT be NULL terminated.
size_stringThe number of characters in the string. (Not including any null terminator). Should be zero if string is NULL.
stringA friendly name to go with the phone number. Maybe NULL.
