Detailed Description

The Device Information server structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the Device Information server library. The elements of this structure are only modified by the Device information Server library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
