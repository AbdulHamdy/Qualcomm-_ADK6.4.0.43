handleConfigFileGetReq

DESCRIPTION
Attempt to open the config definition file on the internal filesystem and send it to the host.
The config definition will be quite large so it will require sending it in more than one message. The AHI task needs to make sure it does not flood the outgoing transport task otherwise the vm will run out of memory to store the queued messages.
