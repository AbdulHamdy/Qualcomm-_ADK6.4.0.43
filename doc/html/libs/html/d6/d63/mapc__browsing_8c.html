◆ MapcMasSetFolderRequest()

void MapcMasSetFolderRequest ( Mas  masSession,
bool  back,
const uint8 *  name,
uint16  len 
)

Request to navigate the folders of MSE.

Parameters
masSessionThe MAS session handle
backTRUE to go backwards before applying the name if exists.
nameThe name of the folder.
lenThe length on the name string.

The API function to navigate the folders of the MSE.

back=TRUE and name=NULL is equivalent to "cd .." or navigate one level up, back=FALSE and name=NULL is equivalent to "cd ~" or navigate to root.

Returns
Message MAPC_MAS_SET_FOLDER_CFM is sent to the application.
