Detailed Description

DESCRIPTION
A set of functions to manage creation, deletion and indexing of multiple data blocks maintained in a single slot.

blockAdd

DESCRIPTION
Add a data block, of a specified size, to the existing pool of data blocks. A pointer to the base of the data block is returned on success, otherwise NULL is returned.
Note: The returned pointer will only be guaranteed to remain valid until a subsequent call to blocksAdd or blockRemove is made.
