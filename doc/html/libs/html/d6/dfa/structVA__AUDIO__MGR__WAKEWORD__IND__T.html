Detailed Description

This is an unsolicitated message which is to indicate that a trigger phrase (wake word) was dectected.

Parameters
start_timestamp: timestamp of the trigger start sample index. Unit of time is in micro seconds
end_index: timestamp of the trigger end sample index. Unit of time is in micro seconds
qva_instance: points to the QVA instance which detected the wake-word. Useful for FFV
trigger_phrase_index: indicates which phrase got triggered (if multiple trigger phrase is enabled)
