Detailed Description

BLE Undirected Advertising Parameters.

This structure contains the Advertising interval max and min range and filtering policy to employ. These are used when the ble_adv_type is OTHER than ble_adv_direct_ind.

For ble_adv_type ble_adv_scan_ind ble_adv_nonconn_ind the advertising interval max range minimum is 0x00A0. If set less, this value shall be used instead.

If NULL default values (indicated below) will be used.
