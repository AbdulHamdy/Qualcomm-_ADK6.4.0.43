Detailed Description

DESCRIPTION
The purpose of this library is to manage the creation and destruction of operators. It sits on top of the operators library and manages any DSP capability bundle file load/unload required. Additionally, it acts as a medium to override capability IDs at a low level, by specifying that you want some capability ID to be replaced with a different one before creating an operator for it.
