33 This is a utility library and provided as a number of
34 functions. It is provided for use by the PBAP library during
35 authentication; performance is not adequate for processing large
36 volumes of data.
37 A typical application will call MD5Init, make a number of calls to
38 MD5Update to supply the data, and finally call MD5Final to extract
39 the data.
40 This library uses the same API as the MD5 reference implementation
41 in RFC-1321, but has been optimised to reduce the code size and
42 memory consumption on BlueCore.
