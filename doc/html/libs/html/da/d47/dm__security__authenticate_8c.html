uint16  timeout 
)

This function is called to authenticate a remote device.

Parameters
theAppTaskThe client task.
bd_addrThe Bluetooth address of the remote device.
timeoutSpecifies the maximum amount of time in seconds for the bonding process. If this time is exceeded then the process is aborted. This value must be set to a minimum of 60 seconds to allow 30 seconds for the IO capability response and 30 seconds for user input, less than this would violate the 2.1 spec. The recommended value for this timeout is 90 seconds.

This normally involves going through the Bonding or Pairing process.
A CL_SM_AUTHENTICATE_CFM message will be sent as a result of this function call.
