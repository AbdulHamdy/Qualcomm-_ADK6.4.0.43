Detailed Description

DESCRIPTION
Library to store file specific information and primarily its file system index. The file entries are organised in groups based on a role.

Get a handle to the file entry with a specific role and number.

Parameters
roleThe role that was used when adding the file entry to the library.
numberSince multiple file entries can have the same role, we need an additional identifier. It is zero based and follows the order in which file entries with the same role were added to the library.
Returns
A handle to access the information stored for that entry. If an error occurred the handle will be a NULL pointer. The function will not cause a panic.

Parameters
