register_custom_sdp

DESCRIPTION
Register a custom SDP record retrieved from PS
The first word in the PS data is the length after unpacking. This allows us to retrieve an odd number of bytes and allows some sanity checking.
Note
This is currently unimplemented for QCC514X_QCC304X platforms
