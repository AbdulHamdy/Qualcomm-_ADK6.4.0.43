Detailed Description

DESCRIPTION
Module talking directly to the Kymera. Chain is a set of connected the Kymera operators.
Sequence of events is:
  1. Turn on DSP
  2. Instantiate a chain
  3. Start a chain
  4. Wait for some event
  5. Stop a chain
  6. Destroy a chain
  7. Turn off DSP
