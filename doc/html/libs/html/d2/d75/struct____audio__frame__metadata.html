rtime_spadj_t __audio_frame_metadata::sample_period_adjustment

The sample period adjustment. A signed fraction in the range ~1.0 to -1.0 that specifies the adjustment to the nominal sample period to give the actual sample period at the time-to-play instant. If no sample period is available this field should be set to 0.0. The actual sample period is calculated as: sample_period = nominal_sample_period * (1 + sample_period_adjustment)
