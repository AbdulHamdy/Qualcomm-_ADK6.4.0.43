Detailed Description

DESCRIPTION
This file contains the implementation of the inquiry management entity. This is responsible for ensuring that only one inquiry request at a time is issued to Bluestack. It also manages the inquiry process by filtering on class of device and initiating remote name requests (if requested by the client).
