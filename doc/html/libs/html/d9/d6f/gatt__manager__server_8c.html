GATT Remote Client Indication Request.

Sends a indication to the client for the handle and value. The client should respond to the indication.

The indication does not update the characteristic value in the server ATT database itself.

GATT_MANAGER_REMOTE_CLIENT_INDICATION_CFM message is returned.

Parameters
taskThe server task that is sending the indication request
cidConnection identifier of the remote device
handleHandle of the characteristic
size_valueLength of the characteristic value to be indicated.
valueThe characteristic value to be indicated. This is copied.
Returns

GATT Remote Client Notification.

Sends a notification to the remote client for the handle and value. There is no response to the application task if this action fails.

The notification does not update the characteristic value in the server ATT database itself.

GATT_MANAGER_REMOTE_CLIENT_NOTIFICATION_CFM_T message is returned.

Parameters
taskThe server task that is sending the notification request
cidConnection identifier of the remote device
handleHandle of the characteristic
size_valueLength of the characteristic value to be notified.
valueThe characteristic value to be notified. This is copied.

Respond to rerouted GATT Access Indication.

Used in response to GATT_MANAGER_SERVER_ACCESS_IND message, indicating that the remote device wants to access a value in the server database and that the application must provide permission for the value to return, depending on the permission flags set.

Parameters
taskThe server task that is sending the access response
cidConnection identifier of the remote device
handleHandle of the value
resultResult of Access Indication.
size_valuethe size of the value (octets)
valueThe value as an octet array, which is copied.
Returns
TRUE if the response was sent, FALSE otherwise.
