Detailed Description

Message containing the local inquiry TX power.

A connection library client may request to read the local inquiry TX power. This message is returned in response to such a request. If the local inquiry TX power was read successfully then the status field should indicate this and the message will contain a valid inquiry TX power.

This message will only be sent if the local device is BT2.1
