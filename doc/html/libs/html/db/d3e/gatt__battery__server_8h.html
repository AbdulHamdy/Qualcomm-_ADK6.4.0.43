◆ GBASS

typedef struct __GBASS GBASS

The battery server internal structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the battery library. The elements of this structure are only modified by the battery library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.

This API is used to notify a remote client of the battery level. This will only be allowed if notifications have been enabled by the remote device.

Parameters
battery_serverThe instance pointer that was passed into the GattBatteryServerInit API.
number_cidsThe number of connection ID values passed in the cids parameter.
cidsThe connection ID list. The battery level notification will be sent on each connection that is supplied in the list.
battery_levelThe battery level to send in the notification.
Returns
