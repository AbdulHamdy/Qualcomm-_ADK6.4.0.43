5 FILE NAME
6  broadcast_stream_service_record.h
7 
8 DESCRIPTION
9  This library defines the Broadcast Stream Service Record (BSSR) and
10  provides functions to parse a BSSR.
11 
12  The BSSR is an array of octets that defines the properties of one or more
13  streams used by a broadcaster. The broadcaster shares the BSSR with a receiver
14  during association. The receiver parses the BSSR to determine whether it
15  supports the streams used by a broadcaster.
16 
17  The stream ID tag must be the first tag defined in a BSSR and is used to
18  delimit the properties associated with each broadcast stream. The properties
19  of a stream may be defined in any order. For example:
20  STREAM_ID_0
21  PROPERTY_OF_STREAM_ID_0_A
22  PROPERTY_OF_STREAM_ID_0_B
23  PROPERTY_OF_STREAM_ID_0_C
24  STREAM_ID_1
25  PROPERTY_OF_STREAM_ID_1_C
26  PROPERTY_OF_STREAM_ID_1_A
27  PROPERTY_OF_STREAM_ID_1_B
28 
29  Each entry in a BSSR is defined using a TAG|LENGTH|DATA structure. In this
30  way a receiver can parse the BSSR, even if it does not understand all the tags
31  defined.
32 */
