Detailed Description

PhoneBook Access Profile Client Library.

Library Dependecies : connection, region, service, obex, bdaddr

Library variants:- pbapc - pbapc with no debug pbapc_debug - pbapc with debug checks pbapc_debug_print - pbapc with debug checks and print output

DESCRIPTION
Interface definition for the PhoneBook Access Profile Client Library.

◆ PbapcPullComplete()

void PbapcPullComplete ( uint16  device_id)

Request to terminate the ongoing Pull operation.

Parameters
device_idPBAPC Session Handle.

This function is to free all resources and complete a pull operation. If the pull operation is already in progress it aborts the operation. Before calling this function, the application must consume any Source data received in the last PBAPC_PULL_PHONE_BOOK_CFM or
PBAPC_PULL_VCARD_LISTING_CFM or PBAPC_PULL_VCARD_ENTRY_CFM message.

Returns
The library returns one of the PBAPC_PULL_*_CFM message on aborting an ongoing pull operation. If there is no pull operation in progress it just free up the resources without returning any message.

Request to terminate the ongoing Pull operation.

DESCRIPTION
Abort an ongoing pull operation or mark the pull operation as complete.
PARAMETERS
Refer pbapc.h

Pull more data from the remote server.

Parameters

This function is to get more data for the ongoing Pull operation if it has returned with a status pbapc_pending. Returns
The library returns one of the PBAPC_PULL_*_CFM message depends on the ongoing pull operation with more data.
