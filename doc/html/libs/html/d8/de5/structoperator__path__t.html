Detailed Description

Describes data path through an operator chain.

Field Documentation

◆ nodes

const operator_path_node_t* operator_path_t::nodes

Pointer to an array of operator_path_node_t structures. Each node identifies an operator by its role, its input terminal and its output terminal. Where the path type indicates that the path does not form input of the chain the input terminal specified by the first node in the array is ignored. Similarly the output terminal specified by the last node in the array is ignored for a path which does form output of the chain.

◆ number_of_nodes

unsigned operator_path_t::number_of_nodes

The number of operators path connects (size of nodes array)

◆ path_role

unsigned operator_path_t::path_role

Role of the path in the chain. This can be any number/enum value as long as it is unique within the chain. The library does not interpret the value in any way. The value is used to refer to the particular path within the chain.

◆ type

path_type_t operator_path_t::type

Type of path. Specifies whether the path forms input and/or output of the chain (Sink and/or Source).
