Detailed Description

DESCRIPTION
Create a broadcast_packetiser configured for the receiver role.
This is only a minimal prototype implementation so that we can get the source of the audio packets and connect it to the sink input of the audio graph.
This code is largely lifted from csb_receiver_packetiser.c but there are many dependencies on the main app that are not always available in an audio plugin. The overall design needs more thought.
