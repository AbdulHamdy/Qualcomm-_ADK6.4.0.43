HandleRebootToResume - In UPGRADE_STATE_REBOOT_TO_RESUME.

DESCRIPTION
Unable to continue with an upgrade as we cannot guarantee the required PSKEY operations.
Once the error that got us to this state is acknowledged we will reboot if the VM application permits, otherwise remaining in this state handling all messages that might otherwise cause an activity.

In most cases there is no error code.
