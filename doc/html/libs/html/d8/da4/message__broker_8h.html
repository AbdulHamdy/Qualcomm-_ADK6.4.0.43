Detailed Description

Interface to the Message Broker.

Copyright
Copyright (c) 2018 Qualcomm Technologies International, Ltd.
All Rights Reserved.
Qualcomm Technologies International, Ltd. Confidential and Proprietary.

The Message Broker allows client Application Modules to register interest in System Messages by Message Group. The Message Broker sniffs Messages sent in the system, and if they belong to the interested Group, forwards a copy to the interested client component.
