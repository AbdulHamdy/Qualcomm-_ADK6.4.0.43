◆ UpgradePSKeys

UPGRADE_LIB_PSKEY UpgradeCtx::UpgradePSKeys

Storage for upgrade library information maintained in PSKEYS.

The values are only read from PSKEY on boot. The local copy can be read/written at any time, but should be written to storage as little as possible.

Since this PSKEY storage structure has multiple uses be aware that any value you write to the local copy may be committed at any time.

The values will always be initialised, to 0x00 at worst.
