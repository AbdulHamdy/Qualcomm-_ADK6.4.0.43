Detailed Description

Interface to the TWS Packetiser library.

The TWS packetiser library contains supports for both master and slave roles.

The TWS master packetiser creates packets containing audio frames read (with metadata) from a kymera Source. The TWS packets are written to a Sink.

The TWS slave packetiser reads packets from a Source. The packets are parsed and the audio frames within a written (with metadata) to a kymera Sink.

Get the packet header length in octets.

Parameters
tpThe instance.
number_audio_framesThe number of audio frames to transport in the packet.
Returns
The header overhead length in octets.

If number_audio_frames is zero, the function will return the number of octets in the primary packet header. For non-zero number_audio_frames, the returned value will also include the number of header octets per frame.
