◆ ConnectionEnterDutMode()

void ConnectionEnterDutMode ( void  )

This function is called to place the device in "Device Under Test" (DUT) mode.

This function places the local device into DUT mode. It makes the device discoverable/ connectible, disables security and then issues the DM_HCI_ENABLE_DEVICE_UT_MODE message to Bluestack. A CL_DM_DUT_CFM message is returned to the task that initialised the connection library to indicate whether DUT mode was entered successfully.
