Filter OUT the capability matching the filter (if any) returning the remaining length. If filtered_caps is supplied then the remaining caps are copied into this. The length needed can be determined by calling the function first with a NULL pointer.

Parameters
filtered_capspointer to memory to take the filtered service caps
