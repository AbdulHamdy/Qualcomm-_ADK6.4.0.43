147 /*
148  @brief Decode a Tag(attribute)/value pair and extract the value.
149 
150  @param element An valid element from the ObexParseTree.
151 
152  @param tag The attribute, whose value to be extracted. e.g:
153  in a op_vobj_element of "TYPE:EMAIL", TYPE is the tag and EMAIL is the
154  value to be extracted. In a op_xml_element of "<msg handle="203050" />"
155  handle is the tag for the value "203050". If this is set to NULL, this
156  API extracts the entire element.
157 
158  @param tagLen The length of the tag.
159 
160  @param maxValLen This is an in-out value. As input, this is the maximum
161  requested length for the value and as output this is the length of the
162  returned value.
163 
164  If there are multiple instance of tag in the element, it returns the
165  first instance and ignore the others.
166 
167  This API allocates memory of length maxValLen and extracts the value to
168  the buffer on return. The application MUST free the returned buffer
169  after processing it.
170 */
