◆ RtimeWallClockEnable()

bool RtimeWallClockEnable ( wallclock_state_t *  state,
Sink  sink 
)

Enable the wallclock for the sink.

Parameters
stateThe wallclock state.
sinkThe sink.
Returns
TRUE if the wallclock was enabled for the sink, otherwise FALSE. Enabling the wallclock starts a background task that periodically calculates the offset between the local system clock and the wallclock. Once the wallclock is enabled for a sink, the functions RtimeLocalToWallClock and RtimeWallClockToLocal may be called.
