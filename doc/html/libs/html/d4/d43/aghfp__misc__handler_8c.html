◆ AghfpSendPhoneNumberForVoiceTag()

void AghfpSendPhoneNumberForVoiceTag ( AGHFP *  aghfp,
uint16  size_number,
const uint8 *  number 
)

Send a phone number to the HF in response to a AT+BNIP request.

Parameters
aghfpA pointer to the profile instance.
size_numberThe number of characters in the phone number.
numberA uint8 array containing one digit of the phone number per array element. The digits are in ASCII format and may include special characters such as '+'. The number should not be NULL terminated.

The application will receive an AGHFP_SEND_PHONE_NUMBER_FOR_VOICE_TAG_CFM to indicate the success of otherwise of sending phone number.
