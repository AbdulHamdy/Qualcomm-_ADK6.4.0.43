89 DESCRIPTION
90  Attempts to validate that a support Service Capabilities list can
91  be parsed and contains reasonable values.
92 
93  This function should allow all valid values, even if the local hardware/software
94  does not support them.
95

97  be dependent on local settings.
98 
99  The 'reconfigure' flag is used to adjust the validation rules depending on if

101 
102  When 'only_check_structure' is TRUE this function only tests that the structure
103  is correct, it does not verify if mandatory entries are present.
104 
105  When 'ignore_bad_serv_category' is TRUE this function does not return an error for
106  service categories that are out of spec.

108  When the function returns FALSE, it will write the service category with the error
109  and the AVDTP error code to error_category and error_code parameters.
110
