bool ConnectionSmGetAttributeNowReq ( uint16  ps_base,
uint8  addr_type,
const bdaddr *  bd_addr,
uint16  size_psdata,
uint8 *  psdata 
)

This function is called to read attribute data from the persistent store, assuming an entry for the device already exists in the paired device list, returning that data immediately.

This function replaces ConnectionSmGetAttributeNow(), which did not support typed_bdaddr addresses. A macro for the ConnectionSmGetAttributeNow() has been added for backwards compatibility that defaults to the TYPED_BDADDR_PUBLIC address type.

Parameters
ps_baseUser persistent data is now stored in the connection library's private PS Keys and so this parameter no longer has any effect. It is kept for backwards compatibility and will be deprecated in the future.
addr_typeThe address type - Public or Random.
bd_addrThe Bluetooth address of the device
size_psdataThe length of the data to be written
psdataPointer to buffer in which data will be returned.

On return, the requested attribute data will be in the psdata pointer, which was passed as a parameter.
