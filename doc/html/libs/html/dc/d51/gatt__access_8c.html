DESCRIPTION
Handles the ATT_ACCESS_IND message containing a queued write (one of many) from Bluestack (flags == ATT_ACCESS_WRITE_COMPLETE).
The stashed GATT_ACCESS_IND, including the whole of the value data, is sent on to the upper layer.
The upper layer is responsible for sending the response.
