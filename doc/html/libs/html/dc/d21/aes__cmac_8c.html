AES-CMAC.

Parameters
key[IN] The secret key
message[IN] The message for which to calculate the MAC
message_len[IN] The length of message
mac[OUT] The message authentication code.

The key and mac should point to arrays of length AES_CMAC_BLOCK_SIZE bytes. The message may be of any length including zero.
