Detailed Description

Debug print functions.

INTRODUCTION

This library allows applications to contain debug printfs in their code and to be able to enable / disable them as required by defining (or not) the following 'switch'.

#define DEBUG_PRINT_ENABLED

PRINT and CPRINT take the same arguments as printf and cprintf.
