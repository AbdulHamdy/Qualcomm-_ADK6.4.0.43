Detailed Description

This message is generate as a result of a call to AvrcpBrowseGetItemAttributesRequest().

On success it returns the number of requested attributes and the attribute Value list. attr_value_list contains list of Attribute Value entries. Each attribute value entry contains a 4 octet attribute Id, 2 octets character set value, 2 octets attribute value length followed by an attribute value string.
