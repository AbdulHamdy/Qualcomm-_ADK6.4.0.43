uint32 TwsPacketiserMasterHeaderLength ( tws_packetiser_master_t *  tp,
uint32  number_audio_frames 
)

Get the packet header length in octets.

Parameters
tpThe instance.
number_audio_framesThe number of audio frames to transport in the packet.
Returns
The header overhead length in octets.

If number_audio_frames is zero, the function will return the number of octets in the primary packet header. For non-zero number_audio_frames, the returned value will also include the number of header octets per frame.
