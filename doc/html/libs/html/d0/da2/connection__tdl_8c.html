Detailed Description

DESCRIPTION
This file contains the functionality to manage a list of trusted devices. A list of devices is maintained in the persistent store.
DM_SM_KEY_ID, used for BLE Privacy, is not supported yet. DM_SM_KEY_SIGN, used for BLE Data Signing, is not supported yet.
