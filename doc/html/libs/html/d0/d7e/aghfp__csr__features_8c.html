Parameters

callerNameHF Device supports the presentation of Caller Names in CLIP and CCWA
rawTextHF Device supports the presentation of raw text from a CSRTXT
smsIndHF Device supports the receipt of SMS indications
battLevelHF Device will send battery level indications (see HfpCsrPowerLevel)
pwrSourceHF Device will send power source indications (see HfpCsrPowerSource)
